Rätsel: Reflected XSS

Finde das versteckte Passwort — mit dem, was du gerade gelernt hast.

Deine Aufgabe

Unten siehst du NeonShop — einen verwundbaren Online-Shop mit Suchfunktion. Die Suchfunktion hat eine XSS-Schwachstelle. Irgendwo im JavaScript der Seite ist ein geheimes Admin-Passwort gespeichert, das nirgendwo sichtbar angezeigt wird.

Ziel

Nutze die XSS-Schwachstelle in der Suchfunktion, um das versteckte Passwort sichtbar zu machen. Gib es dann unten ein, um die Lektion abzuschließen.

Hinweise

Die Suchfunktion nimmt deinen Input und fügt ihn direkt ins HTML ein. Was passiert, wenn dein Input selbst HTML oder JavaScript enthält?

Moderne Browser blockieren <script>-Tags, die über innerHTML eingefügt werden. Aber es gibt andere Wege, JavaScript auszuführen — z.B. über Event-Handler in HTML-Elementen. Denk an onerror, onload, onmouseover...

Versuche diesen Payload in die Suche einzugeben: <img src=x onerror="alert(SECRET_KEY)"> — die Variable SECRET_KEY ist im JavaScript der Seite definiert, aber nirgendwo angezeigt. Dein XSS-Payload kann auf alle globalen Variablen zugreifen.
