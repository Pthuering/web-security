📋 Aufgabe

Du bist Sicherheitsanalyst bei der CryptoGuard AG. Sechs kryptografische Szenarien müssen korrekt zugeordnet werden. Jede korrekte Antwort enthüllt einen Buchstaben. Alle sechs Buchstaben ergeben das Passwort.

Denke an die Kernfragen: Wer hat den Schlüssel? Kann man es rückgängig machen? Wofür braucht man eine CA?
AES = gleicher Schlüssel (symmetrisch), RSA = Schlüsselpaar (asymmetrisch), SHA-256 = keine Entschlüsselung möglich (Hashing).
Die korrekte Reihenfolge der Buchstaben lautet: S, I, C, H, E, R. Ordne die Aufgaben zu und sammle das Passwort.
Aufgabe #1 S
Der IT-Admin will die Festplatten aller Laptops verschlüsseln. Der gleiche Schlüssel wird auf allen Geräten verwendet. Welcher kryptografische Ansatz liegt hier vor?
Aufgabe #2 I
Ein Software-Entwickler möchte sicherstellen, dass eine heruntergeladene Datei auf dem Weg nicht verändert wurde. Er vergleicht den SHA-256-Hash der Datei mit dem auf der Webseite. Was garantiert der Hash hier?
Aufgabe #3 C
Beim Besuch einer Bank-Webseite zeigt der Browser ein grünes Schloss. Welche Organisation hat dem Server-Zertifikat vertraut und es signiert?
Aufgabe #4 H
Wie entsteht eine digitale Signatur für ein Dokument?
Aufgabe #5 E
Eine große Backup-Datei (500 GB) muss schnell verschlüsselt werden. Welcher Algorithmus ist hier die beste Wahl?
Aufgabe #6 R
Im TLS-Handshake sendet der Server nach dem ClientHello sein Zertifikat. Was macht der Client als Nächstes damit?
