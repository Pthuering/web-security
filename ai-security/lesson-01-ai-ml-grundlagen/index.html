Modul 1: AI/ML-Grundlagen für Security

Machine Learning, Transformer, Pipelines — nur so viel Theorie wie nötig, um Angriffsflächen zu verstehen.

Schwierigkeitsgrad
Zeitschätzung
~30 Minuten
Cert-Mapping
SecAI+ Domain 1 (17%)

📋 Voraussetzungen

Keine — das ist die erste Lektion im KI-Security-Track. Grundlegendes IT-Verständnis ist hilfreich, aber keine Voraussetzung.

🎯 Lernziele

  • ML, Deep Learning und NLP voneinander abgrenzen können
  • Die Transformer-Architektur auf Security-relevanter Ebene verstehen
  • Training- und Inference-Pipeline als Datenfluss mit Angriffsflächen beschreiben
  • Tokenization und Embeddings als Konzepte verstehen und erklären können
  • Jeder Pipeline-Stufe mindestens eine Angriffsklasse zuordnen können
