AI-Security Track

AI-Security

LLM-spezifische Bedrohungen verstehen, testen und absichern.

Dieser Track behandelt die OWASP Top 10 für LLMs — von Prompt Injection über Insecure Output Handling bis hin zu Tool Use Security und Agent-Risiken. Die Konzepte bauen teilweise auf dem Web-Security Track auf.

Lektionen

Erste Lektion in Vorbereitung — der Web-Security Track liefert die Grundlagen.
Voraussetzungen

Für den Einstieg empfohlen: Web-Security Lektionen 01–03 (XSS & Input Validation). Viele AI-Security-Konzepte sind Varianten klassischer Web-Angriffe auf LLM-Kontexte.
